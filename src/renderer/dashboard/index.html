    How to use Phayura

    1. Getting started

    1. Click + Add Session in the sidebar to add an account. Name it and pick a group (or leave ungrouped).
    2. Click + Add Group to create a new container window. Each group launches in its own window.
    3. From the group section, click Launch Group (or Launch on a single card) to open the game.
    4. Login on each pane with a different Flyff account. Cookies are isolated per session.

    2. Hotkeys (only inside a group window)

    TabCycle focus across panes
    F11Toggle window fullscreen
    F10Zoom focused pane (hide others); press again to restore

    Hotkeys only switch which pane is focused. They are never injected into the game.

    3. Layout

    • Grid auto-fits the window aspect ratio. Resize the window to change topology.
    • Drag the dark dividers between panes to adjust column/row ratios.
    • Open the session-label dropdown (▽ on the badge) → Edit Position to drag a pane onto another cell to swap.

    4. Drag & drop in dashboard

    • Sidebar: drag sessions to reorder the flat list (group unchanged).
    • Cards: drag onto another card to place before/after; cursor side decides direction.
    • Group section: drop on empty area to move into that group at the end.
    • Ungrouped: drop into the UNGROUPED section to remove from group.

    5. Hover focus (optional)

    Toggle Hover Focus in the top bar. When on, hovering the mouse over a pane focuses it after a short delay (default 30ms). Useful when juggling many accounts without pressing keys.

    6. Reaching the dashboard from inside a pane

    Click the session badge (top-left of any pane) → Manage Panel. Reopens the dashboard if closed.

    7. Compliance

    Phayura never injects input, reads game memory, or attaches a debugger. It only places windows and tracks the cursor (read-only). Safe to use without violating Flyff Universe ToS.
